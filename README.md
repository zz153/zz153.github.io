## Research

My work sits at the intersection of computational genomics and RNA therapeutics, with a focus on glioblastoma (GBM) — one of the most aggressive brain cancers.

- **RNA therapeutics** — antisense oligonucleotides (ASOs), lipid nanoparticle (LNP) formulation
- **Cancer genomics** — lncRNA networks, neoantigen discovery, immune signatures in GBM
- **Multi-omics pipelines** — WGCNA, survival analysis, ceRNA networks across TCGA, GTEx, CGGA
- **Vaccine design** — mRNA influenza vaccine candidate prioritisation using phylogenetics and immunoinformatics

📄 20+ peer-reviewed publications | 🏆 KiwiNet Emerging Innovator Award | 🔬 Co-inventor on patent disclosure

## Projects

### 🥩 [ChurBro](https://churbro.net)
Dunedin's meat price comparison tool. Scrapes New World, PAK'nSAVE, and Woolworths daily using Playwright, normalises prices per kg, and serves them via a static JSON API on GitHub Pages. Built because cheap chops shouldn't require a PhD.

→ [github.com/zz153/churbro](https://github.com/zz153/churbro)

### 🧬 GBM lncRNA Networks
WGCNA-based hub lncRNA identification across 3,369 samples (TCGA, GTEx, CGGA) with survival validation and ceRNA network analysis.

→ [github.com/zz153/GBM_lncRNA_Networks](https://github.com/zz153/GBM_lncRNA_Networks)

### 💉 GBM Neoantigen Pipeline
Rapid neoantigen discovery in IDH-wildtype glioblastoma — integrating somatic mutation calling, VAF-based clonality filtering, and HLA binding prediction via NetMHCpan-4.2.

→ [github.com/zz153/GBM_Neoantigen_Pipeline](https://github.com/zz153/GBM_Neoantigen_Pipeline)

### 🦠 Influenza Vaccine Design
Comprehensive pipeline for influenza vaccine candidate identification across H1N1, H3N2, and VicB lineages using phylogenetics, antigen prioritisation, and immunoinformatics.

→ [github.com/zz153/flu-vaccine-design-distance-analysis](https://github.com/zz153/flu-vaccine-design-distance-analysis)
